# RosieGPT 🌸

RosieGPT is a full-stack AI-powered conversational web application built with modern web technologies and AI integration.

The project features secure authentication, persistent chat history, voice input support, markdown rendering, dark mode, and real-time AI responses.

# ⚙️ Installation & Setup

## 1️⃣ Clone the repository

```bash
git clone https://github.com/omjha-git/RosieGPT.git
```

---

## 2️⃣ Navigate to project folder

```bash
cd RosieGPT
```

---

# 🔧 Backend Setup

## Navigate to backend folder

```bash
cd Backend
```

## Install dependencies

```bash
npm install
```

## Create .env file

```env
OPENAI_API_KEY=your_api_key
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

## Start backend server

```bash
npm start
```

---

# 🎨 Frontend Setup

## Navigate to frontend folder

```bash
cd chatgpt
```

## Install dependencies

```bash
npm install
```

## Start frontend

```bash
npm run dev
```
